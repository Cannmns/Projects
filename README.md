[ Simple Console Calculator (C#) ]

This project is a simple console-based calculator developed to apply the basic principles of the C# programming language. It is one of my first projects in software engineering education.

~ Features
The application can perform the following operations on two integers entered by the user:
- Addition (add): Adds the two numbers.
- Subtraction (sub): Subtracts the second number from the first.
- Multiplication (mul): Multiplies the two numbers.
- Division (div): Divides the first number by the second.

~ Technologies Used
- Language: C#
- Platform: .NET Console Application
- IDE: Visual Studio / VS Code

~ How it Works?
1. When the program starts, it asks you to enter the first number.
2. It waits for you to select the operation you want to perform (`add`, `sub`, `mul`, `div`).
3. It asks you to enter the second number.
4. It prints the result to the screen or gives an "ERROR!" message if an incorrect selection was made.

~ What Did I Learn in This Project?
While developing this project, I reinforced the following basic programming concepts:
- User interaction with `Console.ReadLine()` and `Console.Write()`.
- Data type conversions (String to Integer) with `int.Parse()`.
- Building decision-making mechanisms with `if-else` control structures.
- Variable definition and the use of basic arithmetic operators.
